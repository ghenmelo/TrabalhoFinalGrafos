Procedimento de instalação nos PCs da UFLA

1) executar no terminal: python build/get-pip.py --user
2) executar no terminal: ~/.local/bin/pip install networkx==2.2 --user

Procedimentos para instalação em PC 

1) executar no terminal: curl https://bootstrap.pypa.io/get-pip.py -o build/get-pip.py
2) executar no terminal: python get-pip.py
3) executar no terminal: pip install networkx==2.2
